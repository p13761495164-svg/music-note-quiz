# 识谱练习 iOS

这是可在 Xcode 中签名、测试并提交 App Store Connect 的 iOS 工程。

## 发布前设置

1. 用 Xcode 打开 `MusicNoteQuizIOS.xcodeproj`。
2. 选择 `MusicNoteQuizIOS` Target > Signing & Capabilities。
3. 选择你的 Apple Developer Team。
4. 将 Bundle Identifier `com.ben.musicnotequiz` 改成你账号下唯一的标识。
5. 在真机测试后，选择 Product > Archive，再通过 Organizer 上传。

## 隐私

应用完全离线运行，不联网、不收集数据。练习分数和设置仅保存在应用本机容器中。

## App Store 建议文案

- 名称：识谱练习
- 副标题：随时练习高低音谱号
- 类别：教育 / 音乐
- 隐私政策：不收集任何数据
