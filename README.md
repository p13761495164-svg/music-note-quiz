# Music Note Quiz

Music Note Quiz is a small offline note-reading practice app.

Live app:

https://p13761495164-svg.github.io/music-note-quiz/

It includes:

- `web/` - standalone HTML version
- `ios/` - iOS Xcode project
- `android/` - Android Studio project

## Features

- Practice treble and bass clefs
- Choose from multiple key signatures
- Answer with an interactive piano keyboard
- Save score, clef, key signature, and display settings locally
- Show note name, scale degree, or no label inside the staff note head
